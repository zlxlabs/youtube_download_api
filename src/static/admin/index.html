Cookie 格式说明

Cookie 文件应遵循 Netscape 格式，每行包含 7 个 Tab 分隔的字段：

域名	标志	路径	安全	过期时间	名称	值

示例：

.youtube.com	TRUE	/	FALSE	1735689600	CONSENT	YES+cb

如何获取 Cookie

  1. 使用浏览器扩展（如 Get cookies.txt LOCALLY）导出 YouTube Cookie
  2. 确保 Cookie 包含 .youtube.com 域名
  3. 将内容粘贴到编辑器中并保存
